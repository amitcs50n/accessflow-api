# AccessFlow API

AccessFlow API is a Flask backend project for enterprise data-access governance. The completed Week 1 foundation includes PostgreSQL persistence, database migrations, user registration, password hashing, JWT login, and protected user endpoints.

## Current Features

- Health endpoint
- Organization and user registration
- Secure password hashing
- JWT authentication with token expiration
- Protected current-user endpoint
- PostgreSQL through Docker Compose
- SQLAlchemy models and Alembic migrations
- Marshmallow validation and OpenAPI documentation
- Basic API tests

## Tech Stack

- Python
- Flask
- Flask-Smorest
